Sobre el Proyecto

Este sistema domótico inteligente fue desarrollado utilizando la potente plataforma ESP32, combinando hardware de última generación con software optimizado para crear una solución de automatización del hogar accesible y eficiente.

El proyecto integra 8 funciones principales que cubren las necesidades más importantes de un hogar moderno: desde el cuidado de mascotas y plantas, hasta sistemas de seguridad y monitoreo ambiental.

La comunicación se realiza a través de Telegram, permitiendo control remoto desde cualquier lugar. Además, cuenta con una pantalla OLED para feedback visual y un sistema de alertas sonoras para notificaciones importantes.
